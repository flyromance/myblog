## 数据库mongodb
- window用户
    - 用系统管理员启动git bash，再执行 net start mongo5
- mac
    - mongod --config /usr/local/etc/mongod.conf
- 购买了云服务器后，不用本地启动mongo了


## 开发
- npm run gulp:dev 删除等工作，监听jade模板
- npm run webpack:dev 启动webpack, watch资源
- npm run server:dev 启动sever

## 线上
- npm run build
- npm run server:prod
